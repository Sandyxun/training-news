# 项目更新说明 - 2026-02-09

## 🎉 更新内容

### 1. 修复了原有的两个问题

#### ✅ 问题1：每次只抓取2条信息
**原因**：
- 代码限制每个源只抓取10条
- 时间过滤器只保留7天内的文章

**解决方案**：
- 修改了 `src/rss_collector.py`
- 增加了可配置参数：
  - `max_articles=30`：每个源最多抓取30条
  - `days_back=30`：抓取最近30天的文章
- 可以根据需要灵活调整这两个参数

#### ✅ 问题2：无法抓取中文公众号
**原因**：
- 微信公众号需要通过RSSHub服务
- RSSHub服务在中国大陆访问受限

**解决方案**：
- 改用直接可访问的中文RSS源
- 添加了9个高质量的中文和英文RSS源
- 覆盖了科技、商业、AI等领域

## 🚀 如何使用

### 立即运行采集
```bash
cd /d/projects/training-news
python src/rss_collector.py
```

### 查看结果
采集结果保存在 `data/news.json`，包含：
- 文章标题、链接、摘要
- 发布时间、来源、分类

### 自定义采集参数
如果想获取更多或更少的文章，可以编辑 `src/rss_collector.py`：

```python
# 在文件末尾的 if __name__ == '__main__': 部分
collector = RSSCollector(
    max_articles=50,  # 修改这里：每个源抓取50条
    days_back=60      # 修改这里：抓取60天内的文章
)
```
