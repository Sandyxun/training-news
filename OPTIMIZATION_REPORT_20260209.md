## 🎯 推送策略说明

### 热度判断标准
由于RSS feed不提供阅读量数据，我们使用**发布时间作为热度指标**：

**原理**：
- 📈 最新发布 = 最受关注
- 📈 48小时内 = 热点话题
- 📈 时效性强 = 价值最高

**实现**：
```python
# 按发布时间倒序排序
articles.sort(key=lambda x: x['published'], reverse=True)
# 只保留前5篇最新的
articles[:5]
```

### 精选流程
1. **采集**：抓取48小时内的所有文章（约72篇）
2. **分类**：按4大类别智能分组
3. **排序**：每类按发布时间倒序
4. **精选**：每类保留前5篇最新文章
5. **标记**：为每类的最新文章添加🔥热门徽章
