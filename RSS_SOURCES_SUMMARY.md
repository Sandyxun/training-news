## 采集配置

当前采集参数（可在 `src/rss_collector.py` 中调整）：

```python
collector = RSSCollector(
    max_articles=30,  # 每个源最多抓取30条
    days_back=30      # 抓取最近30天的文章
)
```

## 问题与限制

1. **部分国内源不稳定**
   - FT中文网、第一财经等部分源无法访问
   - 可能需要订阅账号或有访问限制

2. **RSSHub依赖源不可用**
   - 知乎话题、部分微信公众号需要RSSHub
   - 目前RSSHub服务访问受限

3. **英文源访问较慢**
   - ATD、Training Industry等源可能需要较长加载时间
   - 已移除不稳定的英文源

## 建议

1. **当前配置已足够**：9个高质量源，每次可获取100+篇文章
2. **定期检查**：每月检查一次源的可用性
